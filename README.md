# Real-time Streaming of 3 IMUs on IIO + LSL

## Table of Contents
  * [Usage](#usage)
  * [Debug](#debug)
  
## Usage

To use this code, you should first
- Install Lab Streaming Layer (LSL)
- Modify CMakeList.txt about the project name and path to LSL
- Compile with CMake

## Debug
